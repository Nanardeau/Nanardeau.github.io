Description

Par groupe de 4, nous avions pour consigne de faire un site web (sans contraintes) hébergé sur Apache et implémentant une base de données, gérée en PHP. À l'aide des cours de la documentation fournie, nous avons réalisé un site marchand sur le thème du jeu vidéo "Cyberpunk 2077".

Partie facultative

La partie obligatoire était un cours nous faisant manipuler apache sans application directe (manipulation du module PHP, mise en place de page erreur404 personnalisée etc)

Nous avons implémenté une base de données dans laquelle nous avons stocké des articles (des vêtements) ainsi que les informations des utilisateurs pour leur création de compte.
Nous avons également implémenté le fait qu'un utilisateur doive valider son compte avant de pouvoir se connecter avec. Si cela n'est pas fait avant 3h du matin, le compte sera supprimé.
